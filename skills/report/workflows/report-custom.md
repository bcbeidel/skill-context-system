# Custom Report Workflow

<objective>
Generate a custom dashboard for specific date ranges, comparisons, or project milestones.
</objective>

<required_reading>
- `references/dashboard-design.md` - Dashboard principles
</required_reading>

<process>

## Step 1: Determine Custom Parameters

Parse `$ARGUMENTS` or ask user for:

**Date range:**
- Start date
- End date

**OR comparison points:**
- Baseline date/snapshot
- Current date/snapshot

**OR milestone:**
- Project phase
- Release version
- Specific event

## Step 2: Execute Custom Report Script

Run with custom parameters:

```bash
python ${CLAUDE_PLUGIN_ROOT}/skills/report/scripts/generate_report.py --custom --from=[start] --to=[end]
```

Or for comparisons:

```bash
python ${CLAUDE_PLUGIN_ROOT}/skills/report/scripts/generate_report.py --custom --compare=[baseline]
```
